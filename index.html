01. About

I am an aerospace engineering officer in the RCAF as well as a researcher studying the design and implementation of autonomous flight systems for unmanned aerial vehicles.

Recent works have been focused on the use of artificial neural networks in the trajectory planning and guidance schemes of autopilot systems, which has been the subject of a M.A.Sc., available below.

Some of the fields, disciplines, and tools with which I am familiar include:

04. Projects

Deep Learning Soaring

Implemented the proximal policy optimization (PPO) algorithm to train a 3DOF flight model in simulation for autonomous dynamic soaring.
[In progress]

CartPole-RL

Implemented various reinforcement learning algorithms to train controllers for the balancing cart-pole problem. Created simulation environment and visualizations from scratch.
[Report]

Nano-Bot

Created a self-balancing pendulum robot from scratch using an Arduino Nano. Sensors included wheel encoders and a 6DOF IMU, with motors controlled through a PID loop.

MatFlightViz

Created a Matlab-based flight simulation animator for 3DOF flight model trajectories. Takes 3D STL model and aircraft state trajectories and outputs animation in .gif or .avi format.

Web Design

Designed website from scratch using HTML, CSS, and Javascript. Used SQL to manage databases for commercial applications.
